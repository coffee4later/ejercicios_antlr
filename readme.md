Este repositorio contiene los ejercicios del 1 al 10 de los ejercicios de análisis léxico con antlr usando los lenguajes de programación Java y Python.

## Como probar?

Los primeros ejercicios (del 1 al 4) fueron hechos usando python, puedes ejecutarlos con

'''
python nombredelprograma.py
'''

Los ejercicios del 5 al 10 fueron hechos usando Java, puedes probarlos con:

'''
java -cp "$env:CLASSPATH;." ejercicioX
'''

solo modifica los nombres de los respectivos archivos segun el numero del ejerecicio que desees probar.
